A web-based control panel and platform for self-managed, multitenant identity management.

## Dependencies

* Projects covered in [Redwood's introduction](https://redwoodjs.com/docs/introduction#technologies).
* [HeadlessUI (React)](https://headlessui.dev/)
* [i18next](https://www.i18next.com/)
* [PostgreSQL](https://www.postgresql.org/)
* [react-hook-form](https://react-hook-form.com/)
* [RecoilJS](https://recoiljs.org/)
* [TailwindCSS](https://tailwindcss.com/)
  * [Autoprefixer](https://github.com/postcss/autoprefixer)
  * [PostCSS](https://github.com/postcss/postcss)

### Optional Dependencies

LockTech provides configuration for an [API gateway](https://github.com/LockTech/poseidon) and a [relational database](https://github.com/LockTech/portunus).

While not required, the linked repositories may provide use in setting up the application in a development or production environment.
